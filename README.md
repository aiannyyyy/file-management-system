🗂️ NSCSL File Management System

A modern, secure, and scalable file management system built for NSCSL to efficiently organize, upload, share, and manage digital files across the organization.

This system uses a React + TypeScript frontend, a Node.js + Express backend, and a MySQL database, delivering a fast and responsive experience with a clean UI powered by Tailwind CSS.

🚀 Tech Stack
🎨 Frontend

React (TypeScript)

Tailwind CSS

Axios (API communication)

Vite (fast development build tool)

⚙️ Backend

Node.js

Express.js

MySQL

RESTful API architecture

🖥 Desktop Integration

Electron (for desktop application build)

✨ Key Features

📁 File upload, download, and management

🗂 File categorization and organization

🔎 Search and filtering system

💬 Real-time chat module

🔔 Notification system

🔐 Authentication & protected routes

📱 Responsive and modern UI

🖥 Desktop-ready via Electron

⚡ Lightweight and fast performance

🧩 Scalable full-stack architecture
